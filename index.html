HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.


Web browsers recieve HTML documents from a web server or from local storage and render them into multimedia web pages.

HTML describes the structure of a web pages semantically and originally included cues for the appearance of the document.

HTML elements are the buliding blocks of HTML pages. WIth HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
